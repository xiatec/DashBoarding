Data Visualization Portfolio

Welcome to my data visualization portfolio. Below you'll find a collection of interactive Tableau dashboards I've created to transform complex data into actionable insights for various business scenarios.

Greatfit E-Commerce Reviews Analysis

This dashboard was created for Greatfit, an online e-commerce store selling ladies clothing. The Customer Service manager needed visibility into customer reviews across various platforms. The dashboard provides comprehensive analysis of:

  • Positive vs. negative review distribution
  • Review analysis by product division
  • Product recommendation rates
  • Mentions of product returns in reviews

The interactive filters allow the team to investigate negative reviews in detail, helping Greatfit identify improvement opportunities and enhance customer satisfaction.

MegaMart Sales Agent Performance Dashboard

This dashboard was developed for MegaMart, a global retailer that operates through sales agents responsible for specific product categories. The Sales Manager requested a comprehensive tool to monitor agent performance with the following features:

  • Monthly sales tracking for each agent
  • Scatter plot visualization showing quantity vs. sales for each product category
  • Top 10 SKUs by sales for selected month
  • Top 5 product categories by sales performance
  • Sales target tracking functionality

The interactive filters allow management to select specific sales agents and time periods, enabling targeted performance analysis and strategic decision-making.

Park City Peaks Hotel Analytics

This dashboard was created for Park City Peaks Hotel to transform their raw booking data into actionable insights. The hotel manager previously relied on basic Excel spreadsheets downloaded from their booking system but needed more comprehensive analytics to understand:

  • Monthly booking and cancellation patterns
  • Repeat guest statistics
  • Food option preferences among guests
  • Guest composition analysis (individual travelers vs. family bookings)

The interactive visualizations enable management to identify seasonal trends, optimize marketing strategies for different guest segments, and make data-driven decisions to improve occupancy rates and guest satisfaction.

Google Investor Dashboard

This dashboard was designed as a stock monitoring tool for investors, similar to the Facebook stock tracking project in the original brief. It provides real-time and historical stock performance data with key metrics including:

  • Most recent opening and closing prices
  • Percentage change between open and close prices
  • Daily closing price trends
  • Historical performance visualization

The dashboard automatically updates with the latest market data, allowing investors to track Google's stock performance and make informed investment decisions based on current and historical trends.
